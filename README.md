# Information-systems-design

## HealthNavigator - сервер мониторинга здоровья:
- Интерфейс React + Vite, отображающий панель мониторинга с показателями здоровья (частота сердечных сокращений, шаги, качество сна)
- REST API с конечными точками для устройств и показателями жизнедеятельности
- База данных PostgreSQL для хранения данных о состоянии здоровья
- Имитация интеграции носимых устройств (поскольку реальные аппаратные API требуют сопряжения устройств, смоделированы контракты передачи данных таким образом, чтобы к ним могли подключаться реальные устройства)
- Архитектура, готовая к микросервису
